Fonts

Here are a few small fonts I've made in a web app called fontstruct. It's not a good way to make fonts in my opinion, but it is fun to play around with. If you want you can look at and download the fonts from my fontstruct page!

This is a modified version of Jetbrains Mono that I've used as my one and only coding font for a few months at this point. I made it intentionally quirky and weird, but I really like it!

It's also proportional (not monospaced), even though I use it for coding. This is because propotional typefaces are secretly better than monospaced fonts at everything (except ascii art, terminals, markdown, assembly, etc). My point still stands, regular programming should not require a monospaced font, and the uneven color of monospaced type is pretty ugly, so proportional it is.
